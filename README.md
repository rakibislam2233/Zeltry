# Oryzn Client

A modern e-commerce platform built with Next.js, TypeScript, and Tailwind CSS.

## Features

- **Storefront**: Browse products, categories, and manage shopping cart.
- **User Dashboard**: Manage orders, profile settings, wallet, and wishlist.
- **Secure Authentication**: Register, login, and password management.
- **Checkout Process**: Streamlined checkout with order summary.

## Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: React Hooks

## Getting Started

First, run the development server:

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Project Structure

- `src/app`: App router pages and layouts.
- `src/components`: Reusable UI components organized by feature.
- `src/lib`: Utility functions and configurations.
